What is ContenTube?

The goal of ContenTube is to be the best search engine for videos.

Over time, sites like Youtube, Facebook Watch, and Vimeo have driven creators to "play the algorithm" by prioritizing views as the key revenue metric. Hence, methods that generate more clicks have become widespread. Examples of this include:

  1. Titles
    • Clickbait / Fake
    • Trendy / Hot Topic
    • Controversial Take
  2. Description
    • Paid promotion (sponsor ads)
    • Affiliate products / programs
    • Shameless plugs
      • Websites
      • Social media
      • Other videos
    • Unrelated #hashtags

Thus, by limiting search corpora to video titles and descriptions, existing services provide low-quality search functionality for their customers.

In contrast, ContenTube provides an exceptional content-based search, utilizing AI and human-generated transcripts.

How does ContenTube work?

TL;DR

  1. TF-IDF ↔ get the importance of each word in a video
  2. Cosine Similarity ↔ get the similarity of each video for a query

Detailed Explanation

We created ContenTube using techniques and algorithms in the field of Information Retrieval.

Specifically, we utilized the vector space model, where each video and query is represented as a vector in the multi-dimensional space of terms. To obtain these vectors, we implemented a statistical heuristic called TF-IDF (term frequency - inverse document frequency).


When a search is submitted, each video is scored on how well it matches the given query. This is calculated using a measure called cosine similarity.
